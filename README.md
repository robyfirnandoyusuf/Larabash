a 3rd party tool for Laravel with bash

using php artisan without typing the command

Feature : 
- Create MVC
- Migrate 
- Seed 

### Tested on 
Linux Ubuntu 16.04 LTS


### SETUP

- copy & paste larabash.sh to root directory of your project
- chmod +x larabash.sh


### USAGE

- ./larabash.sh
- Choose an option
